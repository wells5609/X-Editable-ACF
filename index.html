X-Editable ACF

X-Editable ACF is a (developer-focused) plugin that allows front-end editing of Advanced Custom Fields in WordPress.

Installation

  • Install and activate plugin
  • Add theme support in functions.php using add_theme_support('x-editable-acf'); (see main plugin loader file)

Documentation

In progress (but don't hold your breath). However, the PHP files are pretty well-documented, if I may say so myself.

Features

  • Edit Advanced Custom Fields from the front-end
  • Currently supports the following fields:
      >
    • Text
    • Textarea
    • Date-picker
    • Taxonomy
    • Number
    • True/False
    • Select
  • Options for displaying a label, values as ul, and more
  • Template tags for easy theme integration
